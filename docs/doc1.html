Local Dev Setup

Install MongoDb, ElasticSearch, ArangoDB

MONGODB

brew install mongodb

Default username password used is admin/admin

use admin
db.createUser(
  {
    user: "admin",
    pwd: "admin",
    roles: [ { role: "userAdminAnyDatabase", db: "admin" } ]
  }
)
If user with admin already exist
db.changeUserPassword("admin", "admin")

arangodb

brew install arangodb

or alternatively: https://www.arangodb.com/download-major/macosx/ install dmg from here. brew version were having issues with starting at default port.

Use UI to setup password. Default used by script is root/root. By default arangodb uses root/

To Start ArangoDB /usr/local/opt/arangodb/sbin/arangod &

Elasticsearch

Create a new directory /mnt1/ /mnt1/config /mnt1/logs then give mnt1 permissions using sudo chmod -R 777 /mnt1/

rename and copy ontic-dev.properties to /mnt1/config/ontic.properties

Do steps for local dev setup in Keyword ssl
